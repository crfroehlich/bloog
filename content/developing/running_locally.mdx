---
title: "Running Locally"
order: 1
---

If you want to develop HikingMyDesk or use plain Gatsby for your HikingMyDesk app
development, then you can follow this guide.

## Download dependencies

```
yarn
```

## Run Gatsby in development mode

```
gatsby develop
```
This will run HikingMyDesk on port 8000 on localhost.

## Build and run docker image locally

```
docker build -t hmd -f docker/Dockerfile .
docker run -dp 80:80 hmd
```
This will run HikingMyDesk on port 80 on localhost.
